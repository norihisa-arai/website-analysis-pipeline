# 05. Excel Output (Transcribe Results)

## Purpose
GPT出力（企業の強み・タグ等）を、所定のExcelテンプレートに自動転記します。

## Variants
- `move_results_v1.py`  … 旧版（`05-1_*` 相当）
- `move_results_v2.py`  … 新版（`05-2_*` 相当 / 推奨）

> **注意**: **どちらか一方** を運用してください。ケースにより精度・使い勝手が異なります。

## Run
```bash
# どちらか一方
python move_results_v2.py
